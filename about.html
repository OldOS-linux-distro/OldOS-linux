About project

What is OldOS?

OldOS is a passion-driven project — a free Linux system designed for old computers. For gamers, for people on a budget, for everyone.

Where did the idea come from?

Many people have old laptops that barely run — too weak for modern systems, too good to throw away. OldOS was created exactly for such machines.

The goal is to create a system that anyone can install in 20 minutes and immediately use — no documentation, no Linux knowledge required.
